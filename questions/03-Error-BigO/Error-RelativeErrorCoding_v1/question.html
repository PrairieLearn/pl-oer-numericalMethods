Suppose in the year 2102 that evaluating $x^{0.72}$ for an input parameter $x$ becomes an important operation to do quickly. As such, CPU manufacturers want to approximate this operation in hardware using a single x86 operation, f_p72. As a computer engineer, you are tasked with creating the necessary components to execute the f_p72 instruction. Suppose that the function f_hat(x) carries out the f_p72 instruction for a given floating point input x. That is, f_hat(x) evaluates the approximation of $x^{0.72}$.

Write a short program that evaluates the magnitude of the relative error in approximating $x^{0.72}$ using f_hat for the input x.

f_hat(x) is a function that returns the approximation $x^{0.72}$ for a given floating point input x. Store the magnitude of the relative error in using the approximation f_hat in relative_error.
