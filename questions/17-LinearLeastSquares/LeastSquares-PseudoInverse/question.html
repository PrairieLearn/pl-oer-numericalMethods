You are given an $m \times n$ matrix ${\bf A}$, with $m > n$. You wish to solve the least-squares problem ${\bf Ax} \cong \mathbf{b}$ for some given right-hand-side vector $\mathbf{b}$. You are also given the SVD $ {\bf A} ={\bf U\Sigma} {\bf V}^T$ of the matrix ${\bf A}$. Which of the following expressions gives the correct least-squares solution to the system?

$\mathbf{V\Sigma^+U}^T\mathbf{b}$ $(\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T\mathbf{b}$ $(\mathbf{A}^T\mathbf{A})^{-1}\mathbf{b}$ $\mathbf{U^{-1}\Sigma}^+\mathbf{V}^{-T}\mathbf{b}$ $\mathbf{U^{-1}V}^T\mathbf{\Sigma}^+\mathbf{b}$ None of these.

The solution of a full-rank least-squares problem is $\mathbf{x} = \left(\mathbf{A}^T\mathbf{A}\right)^{-1}\mathbf{A}^T\mathbf{b}$

If $\mathbf{A} = \mathbf{U}\mathbf{\Sigma}\mathbf{V}^T$ is the SVD of a full-rank matrix, then $\left(\mathbf{A}^T\mathbf{A}\right)^{-1}\mathbf{A}^T = \mathbf{V}\mathbf{\Sigma}^{-1}\mathbf{U}^T = \mathbf{V}\mathbf{\Sigma}^{+}\mathbf{U}^T$

If $\mathbf{A} = \mathbf{U}\mathbf{\Sigma}\mathbf{V}^T$ is the SVD of any (full-rank or rank-deficient) matrix, then the solution to the least-squares problem with minimum 2-norm is $\mathbf{x} = \mathbf{V}\mathbf{\Sigma}^{+}\mathbf{U}^T\mathbf{b}$
