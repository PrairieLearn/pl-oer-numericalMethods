Consider the following initial Python code

A = [1,2,3,4] B = A C = A B.append(5) C = C + [6] B.append(6)

After this code, which of the following Python statements would evaluate to True?

$A == B$ $A$ is $B$ $B == C$ $A == C$ $B$ is $C$ $A$ is $C$
