A computer manufacturer sells motherboards, CPUs, and cases. They only sell products in packages.

  • Package 1 contains {{params.a}} motherboard(s) and {{params.b}} case(s).
  • Package 2 contains {{params.c}} motherboard(s) and {{params.d}} CPU(s).
  • Package 3 contains {{params.e}} case(s).

In total, the company owns {{params.motherboards}} motherboards, {{params.cpus}} CPUs, and {{params.cases}} cases. What unique combination of package sales will allow the company to get rid of its entire inventory?

$\textbf{Hint:}$ You should set up and solve a linear system to find the answer. If you find it helpful, you are welcome to use numpy to solve the system.
